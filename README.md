# 📚 BookHaven — Библиотека

Веб-приложение библиотеки с каталогом книг, системой заказов и административной панелью.

## Возможности

### Пользователь
- Регистрация и авторизация
- Каталог книг с поиском и фильтрацией по жанрам
- Корзина и оформление заказов
- Галерея книг с обложками и кратким содержанием
- Отзывы с фильтрацией (по книге, по зарегистрированным/гостям, по библиотеке)
- Личный кабинет с историей заказов

### Администратор
- Дашборд со статистикой (книги, пользователи, заказы, выручка)
- Просмотр заказов клиентов с фильтрацией по дате и пользователю
- Добавление, редактирование и удаление книг
- Обновление позиций заказов
- Автозаполнение описаний книг из Wikipedia

### Главный администратор (superadmin)
- Все возможности администратора
- Назначение и снятие роли админа у пользователей

## Технологии

- PHP 8+
- MySQL / MariaDB
- HTML/CSS (без фреймворков)
- Wikipedia API (парсинг описаний)

## Установка

```bash
git clone git@github.com:Vantsev/bibloiteka.git
cd bibloiteka
```

1. Убедитесь что MySQL запущен
2. Запустите встроенный PHP-сервер:

```bash
php -S localhost:8000
```

3. Откройте http://localhost:8000

База данных и таблицы создаются автоматически при первом запуске.
